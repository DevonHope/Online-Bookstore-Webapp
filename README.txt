Look Inna Book

DESCRIPTION:
A bookstore that uses a postgresql database to store all current and static information.
This bookstore is currently only available through a terminal, but will have full integration
into a web base platform as well as a desktop application.

This store allows users to browse, buy, create accounts to share their favorite books.

BEFORE RUNNING:
- install python3
- clone or download the git repo as a
- open a command line terminal
